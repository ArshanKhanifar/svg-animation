Java-MicroServices Developer

As a Senior Software Engineer, you will play a central role in the design and development of the services and entitlements platform. The ideal candidate will have a passion for development and a strong background in building dynamic, high-volume domain services/api management platform with Java using an Agile development methodology.

As a fast-growing startup, we look for self-starters who thrive in a fast-paced, agile environment – which means wearing many hats, being able to change direction quickly, and showing an eagerness to learn new technologies as the need arises.

In this role you work with a small, collaborative team of engineers, product managers, and designers – so excellent interpersonal and communication skills are also a must. And most importantly – we look for people that can prioritize, multi-task, and deliver – because it’s a lot more fun to get things done

Must Have Skills:

  • 2+ years of building RESTful services for high volume mobile applications
  • Mandatory experience building a microservice architecture with spring boot and docker, spring boot/cloud, java spark etc.
  • Built microservices (MS) using JSON, HTTP, WebSocket for communicating to external/internal requests.
  • Built service discovery and microservices monitoring to recover from failure and to spread load across more nodes. This includes monitoring of services. Detecting failures. Removing unhealthy nodes out of the mix. Adding additional services into a running system.
  • Experience with reactive programming.
  • Built Java MS which did not require EAR/WAR to deploy.
  • Build Java MS which were deployed using docker, spring boot or other containerization frameworks.
  • Built code which is non-blocking
  • Experience with web technologies (HTML/CSS/JS) is preferred
  • Experience with Node.js and JavaScript frameworks like jQuery are pluses

Tools and Frameworks:

  • Utilize frameworks like QBit to build microservices
  • Utilize redisson, galaxy, hazelcast for in memory data grids.
  • Utilize docker or kubernets to containerize the application and distribute/deploy
  • Other frameworks like, Blade, Dropwizard, Airlift, Ratpack, Restlet, Spring boot etc are nice to have.
  • You must have experience with continuous integration systems (Jenkins)
  • You must have experience unit testing (JUnit, Mockito)
  • You must have experience with version control systems such as Git or SVN
  • Experience working with social service APIs (Twitter, Facebook, Instagram, etc.) is preferred
  • Experience working with Amazon Web Services (AWS) services EC2, S3, etc. is preferred
  • Nice to have zookeepr, consul, denominator for discovery enablement
